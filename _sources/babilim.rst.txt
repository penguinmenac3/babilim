Main Package
============

The babilim package contains everything required to setup the backend and defines the different phases of the training process.

You can change the backend by:

.. code-block:: python

    import babilim
    babilim.set_backend(babilim.PYTORCH_BACKEND)
    # or
    babilim.set_backend(babilim.TF_BACKEND)


For the different phases of training, validation and testing the following constants should be used.

.. code-block:: python

    # Training and Validation
    from babilim import PHASE_TRAIN, PHASE_VALIDATION, PHASE_TRAINVAL

    # Testing
    from babilim import PHASE_TEST
